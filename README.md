#  Proyecto Aerolínea — Programación Avanzada

##  Descripción del proyecto
Este proyecto fue desarrollado como parte del segundo corte de la asignatura **Programación Avanzada** en la **Universidad de La Guajira**.  
Su propósito es implementar los **modelos del caso de uso Aerolínea** usando el framework **Django**, aplicando relaciones entre entidades y configuración del panel de administración.

El sistema permite gestionar:
-  **Aviones** (código y capacidad)
-  **Pasajeros** (nombre, correo y teléfono)
-  **Vuelos** (origen, destino, fecha y avión asociado)
-  **Relación entre vuelos y pasajeros** (Vuelo x Pasajeros)

---

##  Tecnologías utilizadas
- **Python 3.13**
- **Django 5.2.7**
- **SQLite3** (base de datos por defecto)
- **Bootstrap 5** (interfaz del sitio principal)
- **Git / GitHub**
